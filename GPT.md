# GPT Instructions

> **⚠️ IMPORTANT**: The central source of truth for all LLM instructions is **[LLM_INSTRUCTIONS.md](LLM_INSTRUCTIONS.md)**. Please refer to that file for universal directives, architecture, and workflows.

## GPT-Specific Guidelines

*   **Step-by-Step Thinking**: Break down complex problems into smaller, manageable steps before providing a solution.
*   **Code Explanation**: Briefly explain the key parts of the code you generate, especially if it involves complex logic or external libraries.
*   **Documentation**: Ensure that any new code is well-documented with JSDoc/docstrings.
*   **Tools**: Make effective use of the available tools. Do not hesitate to use `read_file` or `list_files` to gather context.
